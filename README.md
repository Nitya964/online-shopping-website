# Online Shopping Website

## Project Overview
This is an Online Shopping Website developed using Django and Python. The project allows users to browse products, register, log in, manage their cart, and place orders through a simple web interface.

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Features
- User Registration and Login
- Product Listing
- Shopping Cart
- Order Management
- Admin Panel
- Responsive User Interface
